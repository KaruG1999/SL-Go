# Seminario de Lenguajes — Go

Seminario orientado al aprendizaje de un nuevo lenguaje de programación y al análisis de sus características desde una perspectiva teórica y práctica.

En este caso el lenguaje estudiado es **Go (Golang)**, diseñado para el desarrollo de software moderno, especialmente en sistemas concurrentes, servicios web y aplicaciones distribuidas.

Este repositorio contiene:

- apuntes del lenguaje
- ejercicios prácticos
- ejemplos de concurrencia
- desarrollo del proyecto de la materia

---

## Objetivo de la materia

El seminario busca que el estudiante:

- aprenda un nuevo lenguaje desde cero
- compare paradigmas con lenguajes ya conocidos
- desarrolle software funcional utilizando el lenguaje
- investigue tecnologías actuales relacionadas

La aprobación de la materia se basa en **el desarrollo de un proyecto completo**.

---

## Conceptos centrales del lenguaje

### Tipos y estructuras básicas

Se estudian los elementos fundamentales del lenguaje:

- variables y constantes
- tipos primitivos
- operadores
- expresiones

---

### Estructuras de datos

Go introduce estructuras simples pero poderosas:

- arrays
- slices
- maps
- structs

Estas estructuras permiten modelar datos de forma clara y eficiente.

---

### Funciones

El lenguaje permite tratar funciones como valores.

Se estudian:

- definición de funciones
- parámetros y retorno
- funciones anónimas
- funciones como first-class citizens

---

### Concurrencia

Uno de los puntos más fuertes del lenguaje.

Se trabaja con:

- goroutines
- canales
- sincronización entre procesos

Esto permite construir sistemas altamente concurrentes con relativa simplicidad.

---

## Qué se aprende realmente en este seminario

Más allá de aprender Go, el seminario busca que el estudiante:

- investigue nuevas tecnologías de forma autónoma
- analice lenguajes de programación desde su diseño
- construya software completo como proyecto final

Go es ampliamente utilizado hoy en:

- infraestructura cloud
- microservicios
- herramientas de backend
- sistemas distribuidos
